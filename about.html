ArmedConflicts.Today: About The Website

Website Info

The purpose of ArmedConflicts.Today is to help inform the public about violent conflicts around the world. There are many wars and conflicts that most people do not know about. This website provides information about the conflicts, related countries, and relevant news.

Data Sources

  • https://en.wikipedia.org/wiki/List_of_ongoing_armed_conflicts: List of conflicts and country information.
  • https://restcountries.eu/: Additional info about countries.
  • https://newsapi.org/: Related news.
  • https://www.charitynavigator.org/ (Charity Navigator): Related charities.

Software Development Tools

  • HTML5, CSS3, Bootstrap: Building the static website.
  • Git, GitLab, Sublime Text, Slack: Tools to aid with creating the software.
  • Amazon Web Services: Hosting for the website.
  • NameCheap: Bought domain from here.
  • Postman: Designing and documenting the RESTful API.
